## 最小生成树概念

### 什么是最小生成树

给定一个连通无向图 `G=(V,E)`，每条边 `e∈E` 都有一个权值 `w(e)`。

生成树的定义：
    它是图的一个子图。
    包含图中所有的顶点。
    是一棵树（无环）。

最小生成树的定义：
    在所有生成树中，边权和最小的生成树。
    也就是说，最小生成树是连接所有顶点且总权值最小的无环图。

### 最小生成树常用算法

常用算法有两种，这里先写一种算法，Kruskal 算法：
    1. 将所有边权从小到大排序。
    2. 初始化并查集，每个点独立。
    3. 遍历排序之后的边，如果边的两个点不在同一个集合之中，则加入MST，再合并集合。
    4. 遍历之后得到 MST。

## 最小生成树代码实现

### 原代码

~~~cpp
#include<bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef long double ld;
const ll mod = 1e9+7;
const ll N = (ll) 3e5+10;
const ll inf = (ll) 1e18;
const double pi = 3.14;

struct DSU
{
    vector<int> parent, cnt;
    DSU(int n)
    {
        parent.resize(n);
        cnt.resize(n, 1);
        for(int i = 0; i < n; i++)
        {
            parent[i] = i;
        }
    }
    int find(int x)
    {
        if(x == parent[x])
        {
            return x;
        }
        return parent[x] = find(parent[x]);
    }
    bool unite(int x, int y)
    {
        x = find(x);
        y = find(y);
        if(x == y)
        {
            return false;
        }
        if(cnt[x] < cnt[y])
        {
            swap(x, y);
        }
        parent[y] = x;
        cnt[x] += cnt[y];
        return true;
    }
};

struct Edge
{
    int u, v, w;
    bool operator<(const Edge &other) const
    {
    return w < other.w;
    }
};

int main()
{
    int n, m; // n = 顶点数，m = 边数
    cin >> n >> m;
    vector<Edge> edges(m);
    for(int i = 0; i < m; i++)
    {
    cin >> edges[i].u >> edges[i].v >> edges[i].w;
    }
    sort(edges.begin(), edges.end());
    DSU dsu(n);
    ll mst_weight = 0;
    vector<Edge> mst_edges;
    for(auto &e : edges)
    {
    if(dsu.unite(e.u, e.v))
    {
    mst_weight += e.w;
    mst_edges.push_back(e);
    }
    }
    cout << "最小生成树总权值: " << mst_weight << "\n";
    cout << "最小生成树的边:\n";
    for (auto &e : mst_edges)
    {
        cout << e.u << " - " << e.v << " 权值:" << e.w << "\n";
    }
    return 0;
}
~~~
